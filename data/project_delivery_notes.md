## Payroll Modernization
The Payroll Modernization project is replacing spreadsheet-based reconciliation with an auditable workflow. Finance owns business acceptance, HR owns employee communication, and IT owns integrations with identity and payroll vendors. The project has a dependency on role cleanup because payroll approvers are still maintained in two systems.

## Checkout Reliability Upgrade
The Checkout Reliability Upgrade was created after incident INC-1023. The work includes connection-pool monitoring, retry storm testing, queue-depth alerts, and a pre-release load-test checklist. The original project brief said "Q2 complete", but the latest steering note moved final validation to early Q3.

## Access Review Automation
Compliance requested automation for quarterly access reviews. The first phase covers finance systems, data export tooling, and production support groups. The project documentation uses both "access review" and "entitlement review", which refer to the same operating control.

## Reporting Caveat
Project budgets in this document are planning figures. Finance reports are authoritative for booked spend, forecasts, and approved budget changes.
